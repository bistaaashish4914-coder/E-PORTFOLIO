JavaScript

JavaScript is the programming language of the web. It adds interactivity, logic, and dynamic behaviour to HTML pages — from simple button clicks to complex single-page applications.

DOM Manipulation
Accessing & Changing the DOM
The Document Object Model (DOM) is a tree representation of the HTML page. JavaScript can select, create, modify, or delete any element in this tree using methods like getElementById, querySelector, and innerHTML.
JAVASCRIPT
// Select an element by ID const heading = document.getElementById('main-title'); // Change its text heading.textContent = 'Hello from JavaScript!'; // Change its style heading.style.color = '#7cffd4'; // Query selector (CSS syntax) const btn = document.querySelector('.submit-btn'); // Create and append a new element const newPara = document.createElement('p'); newPara.textContent = 'I was added by JavaScript!'; document.body.appendChild(newPara);

Form Validation
JavaScript Form Validation
Before submitting data to a server, JavaScript can validate user inputs in real time — checking for required fields, email format, password length, and more. This improves UX and reduces unnecessary server requests.
▶ LIVE DEMO — Form Validation

✅ Form submitted successfully!

JAVASCRIPT
function validateForm() { let isValid = true; const name = document.getElementById('val-name').value; const email = document.getElementById('val-email').value; const pass = document.getElementById('val-pass').value; // Name check if (name.trim() === '') { showError('err-name', 'Name is required'); isValid = false; } // Email format check const emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/; if (!emailRegex.test(email)) { showError('err-email', 'Enter a valid email'); isValid = false; } // Password length check if (pass.length < 6) { showError('err-pass', 'Min 6 characters required'); isValid = false; } if (isValid) { document.getElementById('val-success').style.display = 'block'; } }
