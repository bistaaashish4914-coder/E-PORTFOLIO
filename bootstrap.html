Bootstrap
Framework

Bootstrap is the world's most popular CSS framework. It provides a responsive grid system, pre-built components, and utility classes that dramatically speed up development.

Getting Started
Including Bootstrap via CDN
Bootstrap can be added to any HTML page using a CDN link — no installation needed. This includes both the CSS file and the JavaScript bundle for interactive components.
HTML
<!-- Bootstrap 5 CSS — place inside <head> --> <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet"> <!-- Bootstrap JS bundle — place before </body> --> <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
Grid System
Bootstrap's 12-Column Grid
Bootstrap uses a 12-column grid based on Flexbox. Columns are defined using classes like col-4 (4 of 12 columns) and breakpoint prefixes like col-md-6 (half-width on medium screens and above).

Components
Navbar Component
Bootstrap's Navbar component creates a fully responsive navigation bar with minimal code. It automatically collapses to a hamburger menu on smaller screens.

Utilities
Bootstrap Utility Classes
Bootstrap's utility classes let you apply spacing, typography, colour, display, and flex properties directly in HTML without writing custom CSS.
HTML
<!-- Spacing: m = margin, p = padding, 1–5 = size --> <div class="mt-3 mb-2 px-4">...</div> <!-- Text utilities --> <p class="text-center text-primary fw-bold">Hello</p> <!-- Flex layout utilities --> <div class="d-flex justify-content-between align-items-center"> ...</div> <!-- Hide on mobile, show on medium+ --> <div class="d-none d-md-block">Desktop only</div>
