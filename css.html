CSS

Cascading Style Sheets bring visual life to HTML. From basic selectors to advanced layouts and animations, CSS is what makes the web beautiful and usable.

Selectors & Specificity
CSS Selectors
CSS selectors target HTML elements for styling. Specificity determines which rule wins when multiple rules apply to the same element.
CSS
/* Element selector */ p { color: #333; } /* Class selector */ .card { background: #fff; border-radius: 8px; } /* ID selector */ #header { font-size: 2rem; } /* Pseudo-class */ button:hover { background: #2980b9; } /* Descendant combinator */ nav a { text-decoration: none; }
Box Model
The CSS Box Model
Every HTML element is a box with content, padding, border, and margin. Understanding this model is fundamental to controlling layout and spacing.

Animations
CSS Keyframe Animation
CSS animations use @keyframes to define states and animation properties to control timing, repetition, and direction.
▶ LIVE PREVIEW — Animation
CSS
@keyframes spinPulse { 0%, 100% { transform: scale(1) rotate(0deg); } 50% { transform: scale(1.3) rotate(180deg); } } .box { animation: spinPulse 2s ease-in-out infinite; }
Responsive Design
Media Queries
Media queries allow CSS to adapt based on screen size, making websites responsive and mobile-friendly without needing separate codebases.
CSS
/* Default: 3-column layout */ .grid { grid-template-columns: 1fr 1fr 1fr; } /* Tablet: 2 columns */ @media (max-width: 768px) { .grid { grid-template-columns: 1fr 1fr; } } /* Mobile: 1 column */ @media (max-width: 480px) { .grid { grid-template-columns: 1fr; } }
