HTML

HyperText Markup Language is the skeleton of every webpage. HTML5 introduced semantic elements that make pages more meaningful, accessible, and SEO-friendly.

Document Structure
HTML5 Boilerplate
Every HTML document follows a standard structure. The <!DOCTYPE html> declaration tells the browser we're using HTML5, and semantic elements like <header>, <main>, and <footer> give meaning to the layout.
HTML
<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>My Web Page</title> </head> <body> <header> <nav>...</nav> </header> <main> <section> <h1>Hello World</h1> <p>Welcome to my page!</p> </section> </main> <footer>© 2025 Ashish Bista</footer> </body> </html>
Semantic Elements
HTML5 Semantic Tags
Semantic elements clearly describe their meaning to both the browser and developer. They improve accessibility (screen readers), SEO, and code readability.
<header> <nav> <main> <section> <article> <aside> <footer> <figure> <figcaption> <time>
Working Demo
HTML Form Example
Forms are used to collect user input. Key attributes like type, name, required, and placeholder make forms functional and accessible.
HTML
<form action="/submit" method="POST"> <label for="name">Full Name:</label> <input type="text" id="name" required placeholder="Enter your name"> <label for="email">Email:</label> <input type="email" id="email" required> <select name="course"> <option>BSc IT</option> <option>BSc CS</option> </select> <button type="submit">Submit</button> </form>

Tables & Lists
Structuring Data with Tables
Tables are used for tabular data — not for layout. Use <thead>, <tbody>, and <th scope> for accessible, well-structured tables.
HTML
<table> <thead> <tr> <th scope="col">Name</th> <th scope="col">Technology</th> </tr> </thead> <tbody> <tr><td>React</td><td>Frontend</td></tr> <tr><td>Node.js</td><td>Backend</td></tr> </tbody> </table>

Multimedia
Images, Audio & Video
HTML5 introduced native <video> and <audio> elements. Always include the alt attribute on images for accessibility.
HTML
<!-- Accessible Image --> <img src="photo.jpg" alt="A description of the image" width="600"> <!-- HTML5 Video --> <video controls width="400"> <source src="video.mp4" type="video/mp4"> Your browser does not support video. </video> <!-- HTML5 Audio --> <audio controls> <source src="audio.mp3" type="audio/mpeg"> </audio>
